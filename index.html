Ipsniffer

Sniff the traffic secretly

View project on GitHub

The Tool is a sniffer which sniffs the internet packets in the system and provide a web interface to display the location of origin of packets. The Packets are displayed in their geographical location through a map. The tool was initially made for raspberry pi
